## 进程

### 进程介绍

#### 1.1含义

是操作系统进行资源分配和调度的基本单位，是操作系统结构的基础

一个正在运行的程序或软件是一个进程，程序跑起来就是进程

**注意**：进程里面可以创建多个线程，多进程可以完成多个任务

#### 1.2进程的状态

1. 就绪状态：运行条件都已经满足，正在等待cpu执行
2. 执行状态：cpu正在执行器功能
3. 等待（阻塞）状态：等待某些条件满足，如一个程序的sleep了

```python
print("我是冰冰")#执行状态
sex = input("请输入性别：")#等待状态
print(sex)#执行状态
time.sleep(1)#延迟一秒，等待（阻塞状态）
```
